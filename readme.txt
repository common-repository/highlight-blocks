== Description ==

With this plugin, you can highlight each individual and inside Gutenberg Blocks and display the relative name of the block. which is used in page and post editor. So, the user can easily identify a specific block and update content as per their business needs. It will improve your productivity while updating content on your website.

= Key Features =

* Highlight each individual Gutenberg Blocks and display the relative name of the block. which is used in page and post editor.
* Highlight each inner Gutenberg Blocks. which is used in a specific single block.
* Highlight the separate Gutenberg block using a separate block setting.
* with this toggle button. you can enable and disable block highlight option.

= How it works =

Step 1: Install and active Highlight Gutenberg blocks.
Step 2: To highlight all the block, you need to click on the sidebar setting options.

= INSTALL INSTRUCTIONS =
This section describes how to install the plugin and get it working.

* Upload the folder ‘highlight-block’ to the /wp-content/plugins/ directory of your site.
* Activate the plugin through the ‘Plugins’ menu in wordpress.
